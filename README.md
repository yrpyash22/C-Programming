# C Programming 🚀

A structured collection of C programming examples and practice programs created during my C programming learning journey.

This repository covers the core concepts of the C programming language, starting from the basics and progressing to intermediate and advanced topics. Every program is written for practice, concept building, and future reference.

* Basic Syntax
* Variables & Data Types
* Operators & Expressions
* Conditional Statements
* Loop Control
* Functions
* Recursion
* Arrays
* Strings
* Pointers
* Structures
* Dynamic Memory Allocation
* File Handling (File I/O)

## 🚀 Getting Started

### Requirements

* GCC Compiler (MinGW / GCC)
* Visual Studio Code (Recommended)
* Code::Blocks (Optional)

---

## ▶️ Compile & Run

Compile a program using GCC:

```bash
gcc Basic.c -o Basic
```

Run the executable:

```bash
./Basic
```

For Windows Command Prompt:

```bash
Basic.exe
```
